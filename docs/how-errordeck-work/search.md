# Search

You can search for errors by title & description. 

Later we will add more search options like searching by tags, etc.

## How to search

1. Go to the dashboard
2. Type in the search bar
3. Press enter

## Filter by environment

You can filter by environment to see the errors that are specific to a particular environment. This is useful when you have multiple environments like staging, production, etc.

### How to filter by environment

1. Go to the dashboard
2. Click on the environment dropdown
3. Select the environment you want to filter by

### How to add a new environment

Environment is created automatically when error is reported with a new environment name. You can not create a new environment manually right now.
